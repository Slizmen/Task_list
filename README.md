Простой список задач на python. С встроенной базой данных SQL для сохранения информации о задаче.

В проекте используются библиотеки: PyQt6, sqlite3

main.py - Главный файл.

tasks_design.py - файл с дизайном окна задач.

categories_design.py - файл с дизайном окна категорий.

file - файл базы данных. При запуске программы автоматически создается, если его нет.
